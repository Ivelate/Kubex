Compiled and test version of Monecruft. Can be run using double click.

Options (Command window): 	-fullscreen : Full screen
							-noshadows : Disables shadows
							-map <mapnumber> : Selects map. Map 0: Default map (Islands). Map 1: Mountains map. Map 2: Buggy 3D Noise map.
							

Controls: 	WASD: Movement
			SPACE: Jump
			SHIFT: Enable/Disable flight mode
			Mouse Left Click: Place block
			Mouse Right Click: Delete block
			Mouse wheel: Select block to place / delete
			
Config:

1 game hour = 6 seconds
night is fast forwarded to day (Only lasts 15 sec or so).
